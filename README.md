## Description

Any Website Links Scrapper (puppeteer)

## Installation

```bash
$ pnpm install
```

## Environment

```bash
$ serve a .env file and configure using env.example
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Query Website

```bash
$ Run by giving parameter of any website (i.e. /website?noQuery=true&q=pahe.li)
```
